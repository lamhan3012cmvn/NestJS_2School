## Description

# Clone App

```bash
$ git clone -b dev https://github.com/lamhan3012cmvn/NestJS_2School.git
```

# Run App

## Option 1
## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start:dev
```

## Option 2
Run file run.sh

```bash
$ chmod +x ./run.sh
```
## Running the app

```bash
# development
$ ./run.sh
```
